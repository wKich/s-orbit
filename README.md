s-orbit
=======

Simple 2d orbit calculation tool


================================

How to install:

1. Setup Qt SDK form http://qt-project.org/downloads
2. Clone project and open in Qt Creator
3. Build, Run, Enjoy.

================================

Features

* Bruteforce simulation n-body
* Barnes-Hut simulation
